# 多语言切换功能使用说明

## ✅ 已完成的功能

### 1. 语言配置文件
- `src/locales/zh-CN.json` - 中文翻译
- `src/locales/en-US.json` - 英文翻译
- `src/locales/index.js` - 语言配置导出

### 2. 语言上下文管理
- `src/contexts/LanguageContext.jsx` - React Context 提供语言切换功能
- 支持 localStorage 持久化保存语言偏好

### 3. UI集成
- **导航栏语言切换按钮**：地球图标 + 语言标识（中/EN）
- 点击按钮在中英文之间切换
- 鼠标悬停显示切换提示
- 所有页面文本已替换为翻译函数调用

## 🎯 功能特点

### 即时切换
- 无需刷新页面
- 一键切换中英文
- 流畅的用户体验

### 持久化存储
- 语言偏好保存到浏览器 localStorage
- 下次访问自动应用上次选择的语言

### 响应式设计
- 语言切换按钮在移动端和桌面端都可见
- 简洁的图标设计

## 🔧 技术实现

### 核心架构
```
src/
  ├── locales/              # 语言文件
  │   ├── zh-CN.json       # 中文
  │   ├── en-US.json       # 英文
  │   └── index.js         # 配置
  ├── contexts/
  │   └── LanguageContext.jsx  # 语言上下文
  └── App.jsx              # 主应用（已集成多语言）
```

### 使用方法
```javascript
// 在任何组件中使用
import { useLanguage } from './contexts/LanguageContext';

function MyComponent() {
  const { t, language, setLanguage } = useLanguage();
  
  return (
    <div>
      <h1>{t('nav.features')}</h1>
      <button onClick={() => setLanguage('en-US')}>
        Switch to English
      </button>
    </div>
  );
}
```

### 如何添加新语言
1. 在 `src/locales/` 创建新的语言文件（如 `ja-JP.json`）
2. 复制 `zh-CN.json` 的结构并翻译
3. 在 `src/locales/index.js` 中导入并添加到 translations 对象
4. 在语言切换按钮中添加新的语言选项

## 💡 注意事项

1. **翻译一致性**：确保所有翻译键值都存在于两个语言文件中
2. **默认语言**：项目默认语言为中文（zh-CN）
3. **浏览器兼容性**：使用 localStorage，需要现代浏览器支持
4. **SEO考虑**：当前方案为客户端渲染，搜索引擎只能抓取默认语言
